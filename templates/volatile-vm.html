Volatile VM

Overview

Volatile VM is a software protection solution that makes use of the concept of virtualization to obfuscate the end-users code. It comes with its own instruction set, virtual stack and virtual registers which emulate the workings of an actual CPU. With these handy tools, it will interpret the user code in real time during execution while making sure there are no concerning hits to performance and speed, ultimately protecting your code from otherwise malicious intents.
